### 1. 帶符號大小表示法(signed-magnitude)
```
step1: 判斷正負號(位元最左邊 0為正 1為負)

10010010
↧
負 

step2: 計算大小(magnitude)

0010010
=(1 X 2的4次方)+(1 X 2的1次方)
=(1 X 16)+(1 X 2)
=16+2
=18

step3: 代回正負號
=-18
```
