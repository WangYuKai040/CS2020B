# 4.簡述下列協定的功能與特色並說明他們運作在TCP/IP的哪一層?
(1)HTTP vs HTTPS (2)TELNET vs SSH (3)DNS (4)IP (5)ICMP
```
(1)HTTP(超文本傳輸協定)HyperText Transfer Protocol    vs    HTTPS(超文本傳輸安全協定)HyperText Transfer Protocol Secure
   L7: HTTP是網路通訊基礎，改用HTTPS經加密更安全
 
(2)Telnet(遠端登入)    vs    SSH(安全外殼協定)Secure Shell
   L7: Telnet用於Server遠端控制，明文易遭竊聽，改用SSH經加密更安全

(3)DNS(網域名稱系統)Domain Name System
   L7: 將域名和IP address對映，易於辨識
 
(4)IP(網際網路協定)Internet Protocol
   L3: Packed交換的Protocol
  
(5)ICMP(網際網路控制訊息協定)Internet Control Message Protocol
   L3: 於IP傳送控制訊息,用於返回錯誤資訊或分析路由
```

# 6.簡述下列網路設備主要功能與特色及運作在OSI哪一層
(1)Hub vs Repeater (2)Switch vs Bridge (3)Router vs L3 Switch (4)Proxy
```
(1)Hub(集線器)    vs    Repeater(中繼器)
   L1:  
   hub節點屬同一碰撞網域，易碰撞
   Repeater放大訊號
  
(2)Switch(交換器)    vs    Bridge(橋接器)
   L2:
   Bridge切割碰撞網域，提升頻寬
   Switch每個Port都獨立

(3)Router(路由器)    vs    Switch(交換器)
   L3:
   Router路由與轉送
   L3 Switch依IP位址送Packed

(4)Proxy(代理服務器)
   L7:
   內容過濾、隱藏真實IP，充當防火牆
```
