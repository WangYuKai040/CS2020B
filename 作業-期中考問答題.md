# Communications Protocol(網路傳輸協定)
```
通訊協定（Protocol）是用來溝通不同電腦的一種標準架構。
``` 
```
分層的原則是將來改變或更換某一層的通訊協定時，不會影響到其他層通訊協定的工作。
層次不可太多，太多層的結構在整合的工作上較複雜且困難。
相同功能之通訊協定歸於同一層。
每一層之邊界只與上一層或下一層有關。
```
